First step is to verify the version of AOL you are using.  Bring up 
AOL and go to help and then about.  It should say AOL for Windows 95. 
 If it says AOL for Windows, then you need to sign onto AOL and go to 
keyword UPGRADE and download the file setup95.exe.  Install this file 
and run it (make sure you run the right one as your old AOL is still 
on your HD).  Sign onto AOL, minimize the window and bring up Diablo. 
 Select multi-player, your character then battle.net.  That's it! 
Your on!
