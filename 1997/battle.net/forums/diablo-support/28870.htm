The warrior class maximum magic is 50.
The rogue class maximum magic is 60.
The mage class maximum magic is 250. 
(All the above value is without bonus point like weapon and 
ring...etc)

So the only way is to find a shrine called 'enchanted shrine'. It has 
the effect to RAISE ONE LEVEL to all spells but one (which is LOWER 
by ONE LEVEL) if you played by warrior or rogue. It is called by many 
as 'shrine hunt'.

Hope this help. :-)
