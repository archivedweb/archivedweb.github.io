Does this only happen when you are on battle.net?  If so, follow 
Wargasm's advice (it could be that you are being nuked).

If this happens while not connected to battle.net, you should make 
sure you have the latest possible drivers for your video and (most 
importantly) sound card.  Take a look here: 
http://www.blizzard.com/support/diablo/directx.htm

for a list of links to updated drivers for a number of devices.  If 
your sound or video card is here, follow the link provided, download, 
and install the updated drivers.  Make sure you read any installation 
instructions provided with the download.
