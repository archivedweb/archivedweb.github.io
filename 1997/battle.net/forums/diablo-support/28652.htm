Are you able to play at all on Battle.net?  Somehow I get from your 
message that this problem comes and goes.  If this is the case, you 
need to make sure that your ISP is not restricting TCP packets on 
ports 116 and 118, these ports need to be open for in/out TCP packets.

If this problem has always kept you off of Battle.net, you need to 
upgrade your connection software to 32 bit.  You probably need a 32 
bit winsock.  A good 32 bit connection package is Netscape Navigator 
3.0

http://www.netscape.com

but you can use whatever you want, remember, it has to support 32 bit 
applications.

The Helper

P.S. I know you already said you did this but you have to be 
connected to the internet before you play Diablo also, connected with 
a 32 bit winsock.
