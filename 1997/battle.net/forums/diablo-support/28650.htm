When you are playing Diablo with AOL, you have to connect to AOL 
first, then minimize the connection window, then start Diablo.  You 
also need to start AOL from 

Start - Programs - AOL - AOL for Windows 95

If you start AOL from a shortcut on your desktop, that shortcut is 
probably still pointing at the old version of AOL, verify this by 
choosing help and about AOL, if it does not say its for Windows 95, 
its not the right version.  If you follow this advice you should not 
have any more problems, except for the lag of AOL.
