Mike and Perry,

Take a look for the file updat_cd.com on your hard drive.  This is a 
CD caching program that can cause problems with Diablo.  Use Explorer 
to search for "updat*.*" on your hard drive.  If you find 
this file, rename it.  This will effectively disable it.

If you cannot find this file, then check to see if your CD-ROM drive 
or hard drive are running in DOS compatibility mode:

To see if your CD-ROM or hard drive is running in compatibility mode:

Click on Start -- Settings -- Control Panel
Open System
Select the Performance tab

Win95 will tell you here if you CD-ROM or hard drive is running in 
compatibility mode.

If it is, look at:
CD-ROM
http://www.microsoft.com/KB/ARTICLES/Q151/6/34.HTM

Hard drive.  
http://www.microsoft.com/KB/ARTICLES/Q130/1/79.HTM

For information regarding obtaining the correct drivers for your 
devices.


You should also try using a different read-ahead cache size for you 
CD.  To do this:

click on STart--Settings--control panel
OPen system
select the Performance tab
click on the File Systems button
select the CD tab

There is no hard rule for the optimal size for your CD's read ahead 
cache, so try different settings to see which one works best for you.

If you're still having trouble, download and run the program 
syscheck.exe from our web site here:
http://www.blizzard.com/support/diablo/cd.htm

This will perform a diagnostic on your Diablo CD, and create a 
syscheck.log file.  Mail this file along with a description of the 
problem and your system specifications to support@blizzard.com.
