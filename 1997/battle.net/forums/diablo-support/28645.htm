Prince,

You should check to see if your CD-ROM or hard drive is running in 
DOS compatibility mode:

To see if your CD-ROM or hard drive is running in compatibility mode:

Click on Start -- Settings -- Control Panel
Open System
Select the Performance tab

Win95 will tell you here if you CD-ROM or hard drive is running in 
compatibility mode.

If it is, look at:
CD-ROM
http://www.microsoft.com/KB/ARTICLES/Q151/6/34.HTM

Hard drive
http://www.microsoft.com/KB/ARTICLES/Q130/1/79.HTM

For information regarding obtaining the correct drivers for your 
devices.


If this is not the case, then try using a different setting for your 
read ahead cache.  To do this:

click on start--settings--control panel.
open system
select the performance tab
click on the File systems button
select the CD tab

There is no hard rule for optimal setting for your CD read ahead 
cachesize, so try different ones to see what is the best for you.


If you are still having problems, then download and run the file 
syscheck.exe from our web site.  This file can be found here:
http://www.blizzard.com/support/diablo/cd.htm

This program will perform a diagnostic on your CD, and create a file 
called syscheck.log.  Please mail this file to support@blizzard.com.  

Thanks,
