Danny,

If you are still having problems connecting to battle.net through 
AOL, and you have downloaded and installed the windows 95 version of 
AOL, make sure you are actually running this version when you try to 
connect to battle.net.

When you install the win95 version of AOL, your older version will 
not be deleted and you may still be using this version for connecting 
to AOL.  To make sure you are using the win95 version of AOL, click 
on Start--Programs--America Online--America Online for Windows 95.
