Are you getting any error messages along with your freezes?  You need 
to make sure you have the latest direct X drivers for all your 
devices.  The FAQ at the top of the forum has links to a bunch of the 
different device manufacturers.  You should also make sure you are 
not running any programs that might be interfering with Diablo.  
Before you play Diablo, hit ctrl+alt+del and end task on all programs 
except for Explorer and Systaray.  Then try and play Diablo again.  
If you are still experiencing problems, try turning the sound in the 
game all the way down, see if that works.  If you are still having 
problems please post back and we will try and help you further.

Hope this helps,
