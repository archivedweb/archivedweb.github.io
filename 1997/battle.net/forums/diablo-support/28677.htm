You should manually set your video display to 640x480 and 256 colors 
before you play Diablo.  Do this by going to My Computer - Control 
Panel - Display - Properties - settings.  Make sure you apply the 
changes before you exit, you might have to reboot your computer.  If 
I am right, you were really not in 640x480 256 colors when you 
started Diablo, and when you exited out, the display area was not 
reset to whatever it was set to properly.  I have seen exactly what 
you have described before, because I run in 800x600 256 colors and if 
I get kicked out of Diablo, my display will be in 640x480 and things 
will be mildly screwed up until a reboot.  The funky color scheme you 
saw happens sometimes when errors like this happen.

For this problem, I might be telling you a little more than you have 
to do but if you follow the directions, it will definitely fix the 
problem.

Go to My Computer - Control Panel - Add/Remove Programs - Battle.net 
and remove

After that do the same thing for Diablo, except when you uninstall 
Diablo there will be a little box that it titled also delete saved 
games.  You want to make sure the box is clear if you want to save 
your characters.

After that, go to your Diablo folder, make sure the file BNCACHE.DAT 
is gone, if it is not, delete it.  Also make sure there is nothing 
left in your Diablo folder except for patch executables and your 
single player save game file.

If you do not already have the version 1.04 patch, you should 
manually download it from

http://www.blizzard.com/support/diablo/patch.htm

That file DRTL104.exe goes in your Diablo folder.

Reinstall Diablo, do not play Diablo at this time exit and apply the 
patch.

Run the patch.

Go ahead and try Diablo, your multiplayer options should be back.  I 
am sorry for the long list but I know it will give you back your 
options.  There is a probably an easier way but since no one else is 
answering, thats all I got.
