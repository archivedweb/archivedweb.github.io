wildcard) to diskette.  open MY COMPUTER select CONTROL PANEL select 
NETWORK select VIEW then OPTIONS then IDENTIFICATION and make sure 
that the name of the computer which you are transferring to is the 
same as that which you are transferring from, the computer name is 
case sensitive.  copy the dlinfo_*.drv to diskette then to the 
windows dir on the new computer.  
 To view hidden files select MY COMPUTER, click on VIEW, OPTIONS, 
VIEW, then select SHOW ALL FILES,  select OK.

the ver1.00  to ver1.04 due to problems of losing equipment during
