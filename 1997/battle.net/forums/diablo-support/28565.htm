Both computers have to have the same version of Diablo, if one is 
upgraded, the other has to be also.

You must be using a null modem cable.

You must not be connecting the computers via any other program, 
Diablo should detect the connection on its own.

You gotta make sure that in the device manager of both computers, the 
com ports you are using do not have any conflicts.

There is a text file located in your Diablo folder called serial.doc. 
 That file has a lot of direct connect troubleshooting information.  
If, after checking the text file and the stuff I discussed, you still 
have problems, post back and we will help you further.

Hope this helps,
