Aragon,

Here are things you should check:

-Search for the file "updat_cd.com" on your hard drive.  This 
is a CD caching program that can cause the type of problem you are 
experiencing with Diablo.  Use Explorer to search for the file 
"updat*.*" on your hard drive.  If you find this file, rename 
it.  This will effectively disable it.

-Check to see if your CD-ROM drive or hard drive is running in DOS 
compatibility mode:

To see if your CD-ROM or hard drive is running in compatibility mode:

Click on Start -- Settings -- Control Panel
Open System
Select the Performance tab

Win95 will tell you here if you CD-ROM or hard drive is running in 
compatibility mode.

If it is, look at:
CD-ROM
http://www.microsoft.com/KB/ARTICLES/Q151/6/34.HTM

Hard drive
http://www.microsoft.com/KB/ARTICLES/Q130/1/79.HTM

For information regarding obtaining the correct drivers for your 
devices.


If your devices are not running in compatibility mode, then download 
and run the file syscheck.exe from our web site here:
http://www.blizzard.com/support/diablo/cd.htm

When you run this program, a file called syscheck.log will be 
created.  email this file to support@blizzard.com.  Syscheck.exe will 
performa  diagnostic on your Diablo CD, and its output can help us 
determine what the problem might be.
