Rob,

The porst you mentioned about are the only ones used to connect to 
battle.net and play games, so I don't think this is not a problem 
with your ports.

You say you usually play with 3 people -- are these the same people?  
If they are, then you should make sure that, if they are using a 
modem to dialup to their ISP, they have these modem settings:

Click on Start -- settings -- control panel
Open Modem
click on Modem
select properties
select the connection tab
select Advanced
make sure that Use Flow Control is selected and set it to Hardware
Turn on Error Control -- select Compress Data
Click OK
click on the Port Settings button
click on the default button
click OK

If you are playing with different people, then try to play with 
people with which you have a low latency (the kind of problem you are 
experience can happen in high latency games).  

Thanks,
Pat N.
