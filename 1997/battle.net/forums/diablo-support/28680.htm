If you are trying to connect to Battle.net, you should establish your 
internet connection before playing.  

Make sure your Win 95 modem settings are set to the proper kind of 
dialing you need to use.  Go to My Computer - Control Panel - Modems -
 If you have more than one in the list you can set this for both of 
them.  Select the modem you are gonna check and click dialing 
properties.  Examine the window that comes up, in the bottom of it 
there is an option for tone or pulse dialing, set whichever you want 
to use.  After that you should be set, post back if you have any more 
questions or problems.

Hope this helps,
