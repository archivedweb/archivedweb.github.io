Matt,

To connect to battle.net via AOL, you need to install the win95 
version of AOL.  To do this:

While in AOL, click the Keyword button -- type UPGRADE
you will be able to download the file setup95.exe
after you have downloaded it, run it -- it will install the Win95 
version of AOL onto your system.

After you have the new version:
Connect to AOL using the Win95 version (see the note below)-- 
minimize the window
run Diablo -- choose multiplayer game -- choose your character -- 
choose battle.net

Note: you can check the version of AOL you are using by opening AOL, 
selecting Help from the menu, then selecting About America Online.  
To make sure you are running the Win95 version of AOL, click on Start-
-Programs--America Online--America ONline for Windows 95.
