OK, your problem is not normal, let me explain a little first what 
that error message is....

Your system is not configured for dial up networking, ....  you have 
to make connection manually.  When you start Diablo and try to log 
onto Battle.net, Diablo automatically searches for your internet 
connection, if it does not find an internet connection, it attempts 
to connect to the internet using your Internet Dialer, that is 
configured in Dial up Networking.  That message comes up when it 
detects no connection to the internet and cannot bring up the dialer. 
 Normally this message only appears when you are using AOL 3.0 for 
windows (16 bit application) or when you are using a 16 bit winsock.  
Since you say you are using Netscape 3.0 for your software, when you 
installed it, it should have configured you for dial up networking 
and used that 32 bit winsock that navigator comes with.
