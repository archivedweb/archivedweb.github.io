Goo,

In order to connect to battle.net, all that is required is a 
connection to the internet that supports 32bit applications.  It is 
possible to connect to battle.net using some versions of trumpet 
winsock, though you will have to make sure your version supports 
32bit internet applications (a version of trumpet winsock that 
supports this will probably have a "for Windows 95" in the 
title).

If you have such a connection and still can't connect to battle.net, 
repost here or send email to support@blizzard.com.  Tell us what 
happens when you try to connect (including any error message you 
receive), and we should be able to help you further.
