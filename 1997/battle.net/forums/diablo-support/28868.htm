Helper,

This is true, when AOL users are given this message, this means that 
they are not using the 32bit version of AOL (i.e. AOL for windows 
95), or they have not installed it correctly (though this is more 
rare).

Jason,

I know this is repetitive, but, if you are not absolutely positive 
you have AOL for windows 95, make sure that you are using it.  When 
you run AOL, click on Help, then about America Online.  It must 
actually say in the About window that you are using AOL for Windows 
95.

Also make sure that you have actually connected to AOL before you run 
Diablo.  If you run AOL for windows 95, but do not connect to AOL (by 
clicking on the Connect Now button), then you will get the error 
message you are receiving when trying to connect to battle.net.

If you are connected to AOL and are still getting the 32bit 
applications message when trying to connect to battle.net, then 
uninstall battle.net (not Diablo) using the Add/Remove Programs icon 
in your control panel (to get there, click on Start--Settings--
Control panel).  Make sure the file bncache.dat is deleted from your 
Diablo folder.  Then, connect to AOL again and try to connect to 
battle.net.

If you're still having probelms, see if you are able to run other 
32bit internet applications (like Netscape or Internet Explorer for 
windows 95).  If you are unable to use these, then contact AOL for 
help in installing AOL for windows 95 so that you can use 32bit 
internet applications.
