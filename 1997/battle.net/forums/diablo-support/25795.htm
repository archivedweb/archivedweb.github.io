If you use a program such as Norton AntiVirus or any other program 
which makes a backup copy of the 
registry and have not used it since you reinstalled Win95 you can get 
your old computer name from it. 
Load the file into WordPad and search for ComputerName.  You should 
find it after hitting F3 (find next) once
or twice (1st find won't have the name).

This is the only way I've found to retrieve a forgotten computer name 
for Diablo.

Good Luck, maybe Pat will have some other advise for you don't have a 
backup copy of your registry.
