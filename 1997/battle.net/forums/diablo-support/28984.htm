Jas,

It is possible that you purchases a parellel null modem cable, and 
this would cause you to get the error you received when you tried to 
play the direct connect game.

The cable you successfully used was probably a serial null modem 
cable.  If this is the cable you use for other direct connect games, 
then it is a serial null modem cable.  If it is not too late, try 
returning the cable you recently purchased (and see if it is a serial 
null modem cable, or a parallel null modem cable).
