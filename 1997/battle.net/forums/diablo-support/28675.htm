Before you play Diablo, set your display properties to 640x480 and 
256 colors.  You also need to make sure you have the latest video 
drivers for whatever video card you have.  The FAQ at the top of the 
forum has links to a bunch of the different manufacturers sites.  Try 
changing your display settings first, if that does not work get the 
latest drivers, keep your display settings like that for 
troubleshooting purposes.

If Diablo crashed with some windows error message you need to follow 
the advice I gave you in the first message, you should follow that 
procedure if you ever get a windows error while playing Diablo.

If after all this your display is still messed up, try using Direct X 
2.0, it is on the Diablo CD, some systems work better with it, go 
figure.  I am sorry but I do not know how to install the Direct X 2.0 
drivers, there seems to be no documentation about it.

If that does not work, try direct X 3.0a, it is a little later 
version of Direct X, I am reaching here though, get that at 

http://www.lucasarts.com

Look in the technical support section there under X-Wing vs Tie 
Fighter.
