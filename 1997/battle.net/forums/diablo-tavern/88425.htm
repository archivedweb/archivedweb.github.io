Don't know if this will help but my friend had a similar problem and
the solution was to go into the bios and turnoff the second serial 
port. The modem comandeers one of them so you actually only have one 
available, but the bios didn't know that and kept looking for the 
second port as per tis software and what did it find everytime but 
the printer port. So turning off the second serial port as existing 
in the bios worked for my friend, maybe it would work for yuo too?
