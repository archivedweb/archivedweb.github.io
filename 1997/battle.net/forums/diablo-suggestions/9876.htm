I would like to see a method of looking up multiple players on-line. 
As a guild member I would like to be able to look for a certain name 
pattern so that I can find all members that are on-line when I am. It 
is too hard to /whois every member in the guild. I was thinking of 
something along the lines of "/whois *STC" that would return 
all players with "STC" at the end of their name.

I think this would be an easy feature to implement by simply running 
through all the users on-line and using a regular expression 
evaluator to determine if that player matches the pattern.
