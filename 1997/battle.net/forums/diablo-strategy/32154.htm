When you start diablo, select multiplayer, then create a new 
character.  You can't use your single-player character in multiplayer 
without using a cheat program.  If you want to use the same 
character, surf the net until you find a program which will allow you 
to transfer your single player character to multi.

There IS a difference in equipment, although it is slight.  Some 
items in single player do NOT exist in multiplayer.  I'm not sure if 
the reverse is true.  However, in multiplayer you CAN NOT save your 
games, nor save items which are left laying on the ground.  You MUST 
pick up all of your items before exiting a game or they will ALL BE 
LOST!!

Anyway, I've never played single player actually, but multiplayer is 
a BLAST.
