The tech support forum is a better place to ask this question.

First of all, you need AOL for Win95. To check this: start AOL,
click "Help" in the menu bar, and click "About..." If 
it does not
say it's AOL for Windows 95, then you need to download it from
AOL. Call AOL if you need help setting this up.

Once you have AOL for Win95, connect to AOL, minimize the AOL
program window, start Diablo, and try to connect to battle.net

Hope this helps. If you need more help, post in the tech support
forum.
