I agree that totally random maps would be unusable, but I don't think 
adding  a map generating option to Starcraft would be that difficult.

 All you really need are some structured rules about obstacle 
(mountains and trees)  placement during creation so that the map 
would have a logical layout, and then you could place and evaluate 
each players starting position by their proximity to mines and other 
resources (and other players).  As a final step, a few find-the-path 
checks could be run to make sure there are no "you can't get 
there from here" situations.  If a map created in this fashion is 
unworkable, then the program will junk it and start all over again.  
Hundreds of maps could be created and tested in this fashion very 
quickly.  I would suspect that Diablo uses something to this effect.

Granted, this won't create maps with a specific theme, but I would 
think generated maps could be as playable a pre-created maps.
