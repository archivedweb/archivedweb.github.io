> Hey... It's off topic, but can someone suggest a UVCONFIG setting for

Actually, we suggest not using the built in UVCONFIG program for that 
chip set. To undo the installation of it, delete the UNIVBE.DRV file 
in the \WAR2 folder and set the UNIVBE=1 variable in the WAR2.INI 
file to UNIVBE=0 (11th line, I believe)

S3 has a really nice VESA extension TSR on their website 
http://www.s3.com written by a Dietmar Maschede. Its in their drivers 
sections under the Shareware & Misc section.

You can run the S3VBE20 program in your AUTOEXEC.BAT file so it loads 
in everytime you start up your computer.

If you're confused by that, or need further assistance, give us an 
email at support@blizzard.com

As you noted, this is offtopic :)
