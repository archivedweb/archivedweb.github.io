We already know some info on the Map editor; that it will have better 
AI control and scenario layout. I've also heard a lot of talk of 
adding the ability to make "custom units."   I think that 
might overcomplicate the editor and delay the release. So, here's my 
idea. Make a full-featuredmap editor and then after the game comes 
out make a unit editor seperately. 



The unit editor would be basically graphical. You'd have a big screen 
of all the units and buildings- you could click a button to cycle 
through Terran, Zerg, Protoss, and NPCs/critters/items/scenery. 
Double-clicking on any of them would open a window with all the 
unit's animations displayed at the top, and below a huge array of 
buttons and menus. This would let you change all possible aspects 
that define a unit: can it move, how fast, can it carry other units, 
can it attack, does it fly, what attacks does it use, how powerful 
are the attacks, how frequent can it attack, what are its special 
abilities, and on and on and on. For the graphics you could cycle 
through the frames and paste in one frame at a time. (You would need 
a lot of frames to create a whole new unit graphic.)  You could then 
install the changes in a .pud file.

These abilities would make this a serious tool for editors who have 
real artistic ability

hacked" message.



This would allow purists who don't want any crap to play on the
