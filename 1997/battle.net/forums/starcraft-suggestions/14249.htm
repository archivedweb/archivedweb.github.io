I agree with you here. No way I'm going to hand-draw sixteen views of 
every ship for every view of every animation. This would take MONTHS. 
But it's NOT altogether  impossible. I could create one unit with 
little trouble. I model it in Ray Dream Studio, set up keyframe 
animation for each action (walking, shooting, sitting still, dying), 
render one batch over a bluescreen, rotate 15º, render another batch, 
rotate, repeat 16 times. Then I touch up the views in Photoshop and 
paste them into the editor.



How many people can do this? The majority of people who play games 
don't have the artistic or 3D modeling ability. Others (and this is 
the part that really bothers me) could create really crappy looking 
units that are _obviously_ not from Blizzard. So there would be a lot 
of homemade crap because everyone has this tool but they don't have 
the skills to use it right. 



> 	Again, this is a great idea, however fantastic.  There could be, 

> however, simple ways to edit the races, like replacing the sprites or 

> adding units to existing races, but that falls short of creating a 

> new race.



That's what I said in an earlier post. A single-unit editor would be 
a lot more feasible than a race editor. And if you think about it, 
it's really the same thing. If you replace all the units of a race 
one by one, you'd have in effect created a new race. But no amateur 
game enthusiast has the skills and the time to do that.  Most people 
think "It would be cool to make my own race" but they have no 
idea of the scale of the job they're talking about when they say that.

I think it would be cool to make another ship in a style that was 
clearly Protoss or another building that was obviously Zerg (Hopefuly 
you can tell my reactor is Orcish...) and add it to the game so it 
fit in nicelly with the others.



I'd also like to add that whoever said that they should release a 
unit editor AFTER the game comes out was right. Don't let it delay 
the game by trying to build it into the map editor at the last 
minute. Let us all finish the single-player scenario and have some 
fun on BattleNet and then when the game's newness starts wearing off, 
then spring the finished unit editor on us. Someone might even have 
finished all the frames for one new unit by then.
