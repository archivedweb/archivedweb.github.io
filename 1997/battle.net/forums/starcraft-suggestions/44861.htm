Hello all Starcraft fans and Blizzard,

It would be very cool if you could view the multiplayer map being 
selected, so you can see what the map looks like before you start 
playing it.  What I mean is that when you're in a multiplayer game, 
and you're the game creator, when you're selecting which multiplayer 
map to play, you should be able to see a minimized version of the map 
in the top left or right corner, so that you know what the map looks 
like before you play it.  Maybe even a brief description of the map 
would be nice, like how much gases and metals it has, or what the 
general terrain is like.  If these ideas were to be implemented into 
Starcraft, selecting a map in a multiplayer game would be a very easy 
choice, because then you could know what kind of map you're 
selecting. Thanks for your time,
