I'll look into what help features the game itself will offer, but 
Starcraft will include both inbox and online instructions and 
support.  The manual in the box will be all you need to install and 
play the game (and will include the cool story), but you will also be 
able to come to the Blizzard and battle.net web sites for 
announcements and technical support.

More Starcraft forums will be added to www.battle.net where one can 
receive feedback from other players as well as Blizzard personnel.  
Battle.net itself will add further support for Starcraft (including 
live technical support sessions with Blizzard reps).

This isn't to say you will have technical problems with the game, 
but, if you have a question, many paths to the answer will be 
available :)
