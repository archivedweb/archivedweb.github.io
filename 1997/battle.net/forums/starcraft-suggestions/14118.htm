As mentioned earlier in this forum, I believe that units, as they see 
(and live) through more battle should become more powerful. Like 
"levelling up." If you continously harass the enemy with this 
one Arbiter, or other flying ship (even ground units),  it should 
become more powerful (however, ever so slightly). 

Another thing that would be interesting would be the ability to name 
units. This would be perfectly useless in a battle, but would be a 
neat little addition to the game. However, "unique" units or 
"characters" should not be able to be renamed.

Something else that would be cool, considering there is a maximum 
size of grouping for each race, would be Generals, or perhaps just 
unique characters. The idea would be that in any group that this 
general is grouped with, each unit gains a little plus. This plus 
would be extremely small, and the Generals would be weak, because the 
Zerg would have an advantage because they can group more units 
together.
