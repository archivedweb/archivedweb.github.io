  • Self-starter with ability to work on multiple projects simultaneously
  • Job seekers should be avid gamers and have a passion for working with computers
  • Work full-time, on-site in Irvine, California
Other Useful Skills:
Experience with:
  • Windows NT Server
  • Microsoft Exchange
  • Various Internet applications
  • Virus protection and network security
